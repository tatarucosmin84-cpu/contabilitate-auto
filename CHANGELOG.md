
# ContabilitateAuto — Istoric Modificări

## v1.0.1 — 2026-03-15
### Adăugat
- Dashboard multi-client cu cache inteligent (nu mai rescaneaza la fiecare pornire)
- Tab Declarații cu filtre: status, tip declarație, căutare firmă
- Sortare coloane în toate tabelele (click pe header)
- Filtre în Fișa Detaliată și Verificare Declarații
- Sistem update automat prin GitHub
- Ecran configurare inițială (fără căi hardcodate)
- Modul vector fiscal cu import automat din PDF-uri SPV
- Detector declarații speciale (D390 din D300, D205 din D100)
- Fereastra configurare firme responsive (se adaptează la dimensiunea ferestrei)

### Modificat
- Cache bazat pe amprentă fișiere (funcționează corect pe OneDrive)
- Dashboard cu două tab-uri: Fișă Fiscală | Declarații
